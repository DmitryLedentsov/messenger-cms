<h3>Что делает плагин</h3>
<p>Плагин позволяет генерировать промокоды со скидкой на второй заказ, и отправлять их по почте пользователю. Формирует CRON задание, которое нужно указать в планировщике задач на хостинге. Там же на хостинге указывается время и периодичность выполнения задания.</p>

<h3>Настройка</h3>
<p>При включении плагина создается базовый промокод SECOND-ORDER  на основе которого будут генерироваться одноразовые промокоды</p>
<p>Величину скидки можно изменить в настройках плагина. Также можно задать время действия и задержку оповещения пользователя в днях(по умолчанию 0 - письмо отправляется сразу после успешного оформления первого заказа)</p>
<p>Когда задержка установлена, скопируйте строку с командой и сохраните ее в ежедневную команду CRON(это можно сделать в панели хостинга)</p>
